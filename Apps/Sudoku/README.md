# 🧩 Logic Sudoku for Seniors
> **Built with SpriteSystem OS (Hybrid Workflow v1.6.0)**

---

## 🛠 開発プロセス：Hybrid & Iterative Workflow
AIモデルの特性を最適に組み合わせ、人間による最終調整を経て完成した、SpriteSystemの代表的なプロダクトです。

### 1. Phase 1: Architecture (Flash)
軽量・高速な SpriteSystem_Flash を使用。シニア向けアクセシビリティ（アイボリー基調のデザイン）と、基本データ構造のプロトタイプを迅速に構築。

### 2. Phase 2: Logic Design (Pro)
構築された仕様書を SpriteSystem_Pro に投入。ビット演算（Bitmask）による高速判定エンジンと、人間的な思考を模倣する「論理的思考シミュレータ」の実装。

### 3. Phase 3: Final Polish (Human)
実機テストに基づき、レスポンシブ配置や十字ハイライトの視認性を微調整。さらに、家族への感謝を込めたイースターエッグ（特定日のお祝い機能）を直接統合。

## ✨ 主な特徴 (v1.6.0)

### 🎨 シニア特化型デザイン
- **視覚保護**: 純白を排除したアイボリー（#F5F5DC）背景により、白内障や視覚過敏の方でも長時間プレイ可能。
- **高視認性**: 濃紺（#000033）の極太フォントと、選択マスが分かりやすいグレージュ（#E0E0C8）の十字強調を採用。

### 🧠 次世代生成エンジン
- **論理性保証**: 単なる全探索ではなく、Naked Single/Hidden Single等の手筋で解ける「良問」のみを生成。
- **超高速判定**: 1ファイル完結型ながら、ビット演算により入力ミスを瞬時に判定。
- **安全設計**: 生成処理に 2000ms の制限を設け、古い端末でのフリーズを徹底防止。

### 📦 究極のポータビリティ
- **シングルファイル構成**: HTML/CSS/JSを1つのファイルに完全統合。外部ファイルへの依存一切なし。
- **通信不要・サーバー不要**: 転送されたファイルをブラウザで開くだけで、オフライン環境でもそのまま動作。
