# 🧩 Logic Sudoku for Seniors
**Built with SpriteSystem OS**

このアプリは、自作AI設計基盤「SpriteSystem」を用いて開発された、アクセシビリティ特化型のナンプレ（数独）アプリケーションです。

## 🌟 プロジェクトの目的
「技術を大切な誰かのために役立てる」ことを実証するため、手術後のシニア層や視覚過敏を持つユーザーが、ストレスなく論理的なパズルを楽しめる環境を構築しました。

## 🛠 技術的ハイライト

### 1. SpriteSystem による論理設計
本アプリは、自然言語で書かれた「仕様書」から、破綻のない複雑なロジックを生成する SpriteSystem の能力を証明しています。
- **ヒューマン・ロジック・エンジン**: AIが単に盤面を作るだけでなく、人間が解く際の定石（Naked Single等）に基づいた難易度を自動算出。
- **セーフティ設計**: アルゴリズムの無限ループを防ぐ「2000ms タイムアウト」を標準実装。

### 2. アクセシビリティ・デザイン
- **視覚過敏への配慮**: 目に刺さる純白を避け、温かみのあるアイボリー（#F5F5DC）を基調とした配色。
- **ユニバーサル設計**: 手の震えや誤操作を防ぐ大きなボタン配置、リアルタイムのバリデーション機能。

### 3. 遊び心（イースターエッグ）
- **パーソナライズ実装**: 家族への感謝を込め、特定の日付にのみお祝いメッセージが表示される隠し演出を組み込んでいます。これは、AIがユーザーの「想い（コンテキスト）」を正確にコードへ反映できる柔軟性を持っていることを示しています。
