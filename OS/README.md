# SpriteSystem OS v10.3 | Logic Core Spec

本ディレクトリには、SpriteSystem OS の核となるシステムプロンプト・ファイルが格納されています。

## ■ v10.3 核心機能 (Advanced Logic)

本バージョンの論理エンジンには、以下の高度な制御機構が実装されています。

* **PHASE 0: SELF-DIAGNOSTIC:** 回答開始前に、AI自身に指示の理解度をセルフチェックさせ、論理の脱線（ドリフト）を未然に防ぎます。
* **DRIFT DETECTION:** 会話が長期化した際の、指示の忘却やブレを自動的に検知・警告します。
* **RELOAD TRIGGER:** システムが不安定になった際、特定のコマンドで論理構造を強制的にリセット・再構築します。
* **ADAPTIVE DIFF STRATEGY:** 修正量に応じて「全置換」と「部分修正」を自動判別し、トークン効率を最適化します。

## ■ ハイブリッド運用ガイド

本OSは、Gemini 3.0 の各モデルの特性に合わせて使い分けることで真価を発揮します。

1. **SpriteSystem_Pro.md**
   - **用途:** 複雑な論理設計、高度な分析、厳密なコード生成。
   - **強み:** 深い推論能力を最大限に活用し、一貫性を極限まで高めます。

2. **SpriteSystem_Flash.md**
   - **用途:** 日常的な対話、高速なプロトタイピング、大量のテキスト処理。
   - **強み:** 圧倒的なレスポンス速度とトークン効率を維持しつつ、OSの論理枠組みを適用します。

## ■ 開発サイクル推奨フロー

1. **【特化】Flashモードでの試行錯誤**
   AIを目的の方向に特化させ、正解の糸口を掴みます。
2. **【実行】Proモードによる高度な出力**
   分析や正確なコード出力が必要なフェーズで移行し、重量級の作業を完遂させます。
3. **【資産化】テンプレート化**
   成功したプロセスを「外付けテンプレート」として保存し、OSのクリーンな状態を維持します。
