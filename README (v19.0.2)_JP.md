## ❖ SpriteSystem とは？
**SpriteSystem (OS)** は、LLM（大規模言語モデル）の推論プロセスを構造的に制御し、論理の崩壊（ハルシネーション）を物理的に封じ込めるための **プロンプト・オペレーティングシステム** です。

「AIが言うことを聞かない」「長文になると論理が破綻する」という課題を、法段階説に基づく **RANK 0-9 の階層構造** と、独自の **癒着修正（Semantic Spacing）** 技術によって解決します。

## ❖ なぜこの OS が必要なのか
2026年、AIの知能は飛躍的に向上しましたが、同時に「嘘を吐く」「指示を忘れる」といった制御の難しさも増しています。SpriteSystem は以下の機能で AI を「完全統治」します。

* **Dual-Core Dynamic Gearing**: 重厚な推論 [TITAN] と高速処理 [AERO] を自動切替。
* **Logic Drift Guard**: 文脈の「滑り」を検知し、ハルシネーションをリアルタイムで抑制。
* **Linguistic Integrity**: トークンの「癒着」を防ぎ、AIの注意力を1文字単位で固定。

## ❖ 使い方 (Setup)
1.  **Google AI Studio** を開く。
2.  `System Instructions` に、本レポジトリの `SpriteSystem_v19.0.2.txt` の内容を貼り付ける。
3.  Model で `Gemini 1.5 Pro` または `3.1 Pro Preview` を選択。
4.  `Temperature: 0` に設定。
5.  チャット欄で `/reboot` と入力し、システムを初期化。
