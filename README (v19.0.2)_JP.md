## ❖ SpriteSystem とは？
**SpriteSystem (OS)** は、LLM（大規模言語モデル）の推論プロセスを構造的に制御し、ハルシネーション（捏造）を物理的に封じ込めるための **プロンプト・オペレーティングシステム** です。

専門家による複雑な命令の羅列ではなく、日常的な「使い勝手の良さ」を追求した結果生まれた、法段階説に基づく **RANK 0-9 の階層構造** が、AIを「気まぐれな助手」から「精密な論理演算機」へと変貌させます。

### 核心的機能
* **RANK 階層構造 (0-9)**: 倫理、普遍的論理、現実への錨（RANK 0-3）を最優先。ユーザーの指示（RANK 8）よりも規律を上位に配置し、AIの「暴走」を構造的に阻止します。
* **Dual-Core Dynamic Gearing**: 重厚な深層合成を行う [TITAN Engine] と、高速度で実行する [AERO Engine] をタスクの難易度に応じて自動的に切り替えます。
* **Logic Drift Guard**: 文脈の「滑り」を検知し、AIに1分以上の長考を強いることで、安易な回答（パンダリング）を拒絶し真理へ到達させます。
* **Linguistic Integrity (癒着修正)**: トークンの「癒着」を物理的に防ぎ、AIの注意力を1文字単位で固定する独自のスペーシング・フォーマットを採用。

## ❖ セットアップ (Usage)
1. **Google AI Studio** にアクセスします。
2. `System Instructions` 欄に、本レポジトリ内の OS ソースコードを貼り付けます。
3. **Model**: `Gemini 3.1 Pro Preview` または `Gemini 1.5 Pro` を選択。
4. **Parameters**: `Temperature: 0` / `Safety Settings: Off` (安全管理は OS の Kernel Shield が行います)。
5. チャット欄で `/reboot` と入力し、システムバナーが表示されれば準備完了です。
