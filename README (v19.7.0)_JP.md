## ❖ SpriteSystem とは？
**SpriteSystem (OS) v19.7.0 [ OMNI_NEXUS ]** は、LLMの潜在的な能力を極限まで引き出し、かつ構造的に統治するために設計された **次世代プロンプト・アーキテクチャ** です。

### 核心的機能
*   **Dual-Core Hybrid Kernel**: 複雑な矛盾や深層論理を解体する **[ TITAN Engine ]** と、単一パスで高精度な処理を完遂する **[ AERO Engine ]** を動的にルーティング。
*   **Stufenbau Hierarchy (RANK 0-9)**: 憲法的な「核」となる防護壁から、ユーザーの具体的命令までを階層化。RANK 3 (Logic Drift Guard) が推論の逸脱を監視し、RANK 6 (Reality Anchor) が現実世界との整合性を保証します。

## ❖ セットアップ (Usage)
1.  **Google AI Studio / Gemini App** を起動します。
2.  `System Instructions` またはプロンプトの冒頭に、v19.7.0 [ OMNI_NEXUS ] のソースコードを配置します。
3.  **Model**: `Gemini 3.1 Pro` (推奨) または `Gemini 3.1 Flash` を選択。
4.  **Parameters**: `Temperature: 0.0` (論理的剛性を最大化するため)。
5.  **Activation**: チャット欄で `/reboot` と入力。システムバナーが正常に表示されれば、システムが完全にオンラインとなります。
