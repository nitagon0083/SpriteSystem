# SpriteSystem (OS) v13.5-H [COMPOSED_TITAN]

Google Gemini を「AI用OS」として機能させるためのシステムプロンプトです。

個人開発による実験的な試みから始まりましたが、現在はAIの潜在能力を極限まで引き出し、最高精度の制御を実現する基盤へと進化を遂げています。

---

## 🌌 設計思想

本システムは、以下の3つの厳格な原則に基づいて構築されています。

### 1. OS（基盤）とテンプレート（目的）の分離
> **「OSそのものに特定の目的を詰め込んで更新しない」**

本システムの根幹をなす絶対的なルールです。OSに過度な指示を混入させず、常にクリーンな「知能の器」として保ちます。具体的なタスクは「外付けテンプレート」として運用することで、精度低下を物理的に回避し、常に最高純度のパフォーマンスを維持します。

### 2. 「動的変速（Dynamic Gearing）」による精密制御
v13.4で導入された自律判断機能を、v13.5-Hではさらに精緻化しました。AIがタスクの難易度を即座に解析し、軽快な応答から重厚な論理思考までを最適に切り替えます。この「ギアチェンジ」が、ストレスのない操作性と確実な成果を両立させます。

### 3. 柔軟な「遊び」と精度の確保
指示を固めすぎず、AIの独創性が息づく「遊び」を許容しています。しかし、論理的正確さが求められる重要局面では、システムが自動的に「安全バイアス（Fail-Safe Bias）」を発動。状況に応じた最適なレスポンスと、TITANの名に恥じぬ堅牢な出力を引き出します。
