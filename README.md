# SpriteSystem

Google Gemini を「AI用OS」として機能させるためのシステムプロンプトです。

初心者が個人で開発している実験的な試みですが、AIが本来持っている能力を最大限に引き出すことを目的としています。

---

## 🌌 設計思想

本システムは、以下の3つの考え方に基づいて設計されています。


### 1. OS（基盤）とテンプレート（目的）の分離

**「OSそのものに特定の目的を詰め込んで更新しない」**
これが、本システムで最も重要なルールです。

OSに多くの指示を詰め込みすぎると、AIの焦点がぼやけ、精度が低下します。
基盤となるOSは常にクリーンな状態を保ち、特定のタスクは「外付けのテンプレート」として運用することで、高いパフォーマンスを維持します。


### 2. 対話によるテンプレートの共創

あらかじめ用意された指示を与えるだけでなく、OSとの対話を通じて、その時の目的に最適な「指示の型（テンプレート）」をAIと一緒に作り上げる運用を想定しています。


### 3. 柔軟な「遊び」による精度の確保

指示をガチガチに固めすぎると、AIの柔軟性が失われ、性能が十分に発揮されないことがあります。

あえて指示に少し「緩さ（遊び）」を持たせることで、AIが状況に応じて柔軟に対応できるよう配慮しています。

## 🚀 使い方

1. **導入**
   `Pro` または `Flash` の内容をコピーし、チャット欄や System instructions に貼り付けます。

2. **起動**
   OSが読み込まれたら、あなたのやりたいことを伝えてください。

3. **運用**
   **OSそのものは書き換えずに**、対話しながら最適なテンプレートを作成・実行してください。
