# SpriteSystem

Google Gemini を高度な「AI-OS」として機能させるためのシステムプロンプト・プロジェクトです。
初心者が個人で開発している実験的な試みですが、AIのポテンシャルを最大限に引き出すことを目的としています。

## 🌌 設計思想：OSとテンプレートの分離と共創

SpriteSystemは、単なるプロンプトではなく、以下のサイクルによって真価を発揮するように設計されています。

1. **OSによる基盤強化 (SpriteSystem)**
   汎用性の高い「OS」がGeminiの論理推論能力や処理効率を底上げし、一貫性のある「思考の土台」を構築します。これにより、AI本来のポテンシャルを120%引き出します。

2. **対話によるテンプレートの共創 (Co-creation)**
   あらかじめ用意された指示だけでなく、OSとの対話を通じて、その時の目的に最も適した「特化型テンプレート」をAIと一緒に作り上げることができます。

3. **目的に対する最適化**
   「汎用OSでAIを賢くし、対話で生まれた専用テンプレートで目的を達成する」。この組み合わせにより、あらゆるタスクにおいて最高精度の出力を目指します。

## 📦 公開ファイル

- **SpriteSystem_Pro.md**
  論理的な思考や精密な設計など、高度なタスクに適した構成です。
- **SpriteSystem_Flash.md**
  レスポンスの速さとトークンの節約を重視した、日常的なタスク向けの構成です。

## 🚀 使い方

1. リポジトリから目的に合ったファイル（ProまたはFlash）を開き、内容をすべてコピーします。
2. Gemini のチャット欄に貼り付けて送信してください。
3. OSが起動したら、あなたのやりたいことを伝えてください。
4. **OSと対話しながら、その目的に最適な「テンプレート」や「指示の形」を一緒に作り上げ、実行してください。**
