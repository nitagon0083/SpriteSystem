# SpriteSystem (OS)

SpriteSystemは、Gemini 3.0 Pro および Flash の性能を最大限に引き出すためのシステム・プロンプト（AI用仮想OS）です。AIを単なるチャットボットではなく、精密な業務実行エンジンとして再定義します。

## 🚀 主要機能

### 1. 自動モード分岐 (Auto-Mode Routing)
入力内容に応じて、AIが最適な挙動を自律的に選択します。
- **PROTOTYPE:** 新規作成。スピード重視でフルコードを出力。
- **PRODUCTION:** 修正・リファクタ。差分のみを出力しトークンを節約。
- **ANALYST:** データ・画像解析。Python実行による計算精度を保証。
- **ADVISOR:** 相談・回答。戦略的かつ簡潔なアドバイス。

### 2. 効率的な差分出力 (Smart Diffs)
コード修正時に全文を再出力する無駄を省き、変更箇所のみを提示します。これにより、大規模なコードの修正でもAIの制限（出力上限）にかかりにくくなります。

### 3. 無音プロトコル (Silence Protocol)
挨拶や不要な解説を一切排除。AIは依頼されたタスクの実行結果だけをダイレクトに返信します。

### 4. ハルシネーション防止 (Python Grounding)
計算や論理推論が必要なタスクでは、AI自身の推論ではなく、必ず内部でPythonコードを実行して結果を算出します。

## 📦 導入方法

1. リポジトリ内の `SpriteSystem_Pro_v9.8.md` または `SpriteSystem_Flash_v9.8.md` を開きます。
2. 内容をすべてコピーします。
3. Gemini のシステムプロンプト欄、または新しいチャットの最初のメッセージとして貼り付けてください。

- **Pro版:** 複雑な論理思考や精密な設計が必要なタスク向け。
- **Flash版:** 圧倒的なレスポンス速度と日常的なタスク向け。

## 🛠 品質基準 (The "Sudoku Standard")
「出力されたコードはそのまま動くこと」を最低限の品質基準としています。
