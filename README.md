# SpriteSystem

Google Gemini を高度な「AI-OS」として機能させるためのシステムプロンプト・プロジェクトです。
初心者が個人で開発している実験的な試みですが、AIのポテンシャルを最大限に引き出すことを目的としています。

## 🌌 設計思想：OSとテンプレートの分離・共創・柔軟性

SpriteSystemは、以下の3つの柱によって真価を発揮するように設計されています。

### 1. OSによる基盤強化 (SpriteSystem)
汎用性の高い「OS」がGeminiの論理推論能力や処理効率を底上げし、一貫性のある「思考の土台」を構築します。これにより、AI本来のポテンシャルを120%引き出します。

### 2. 対話によるテンプレートの共創 (Co-creation)
あらかじめ用意された指示だけでなく、OSとの対話を通じて、その時の目的に最も適した「特化型テンプレート」をAIと一緒に作り上げます。

### 3. 精度の維持と柔軟な「遊び」
- **分離の重要性:** OSとテンプレートをあえて統合せず分離して運用することで、命令の肥大化による焦点のぼやけや精度低下を防ぎます。
- **柔軟性の確保:** 指示をガチガチに固めすぎるとAIの性能に不安が生じるため、あえて少し「緩さ（遊び）」を持たせた設計にしています。これにより、AIが指示に縛られすぎず、状況に応じて柔軟にポテンシャルを発揮できる環境を維持しています。

## 🚀 使い方（導入方法）

お好みの方法で `SpriteSystem` の内容（ProまたはFlash）を適用してください。

- **方法A：チャット欄に直接貼り付ける**
  新しいチャットを開き、内容をそのまま貼り付けて送信します。
- **方法B：System instructions に設定する**
  Google AI Studio 等の「System instructions」欄に貼り付けることで、常にこのOSが起動した状態で会話を開始できます。
- **方法C：Gem (カスタムGemini) として作成する**
  Gemの作成画面で「指示」として貼り付けます。自分専用の「SpriteSystem搭載Gem」として保存でき、いつでも呼び出せます。

OSが起動したら、やりたいことを伝えてください。**OSと対話しながら、その目的に最適な「テンプレート」を一緒に作り上げ、実行してください。**
