# SpriteSystem

Google Gemini を高度な**「AI-OS」**として機能させるためのシステムプロンプト・プロジェクトです。
※初心者が個人で開発している実験的な試みですが、AIのポテンシャルを最大限に引き出すことを目的としています。

---

## 🌌 設計思想：OSとテンプレートの分離・共創・柔軟性

SpriteSystemは、以下の3つの柱によって真価を発揮します。

### 1. 基盤強化（OS）と目的特化（テンプレート）の分離
* **【最重要】統合して更新しない**
    OS側に特定の目的を詰め込みすぎると、AIの焦点がぼやけ、処理精度が著しく低下します。
* **クリーンな運用**
    基盤（OS）は常にクリーンな状態を保ち、目的ごとの指示は「外付けのテンプレート」として運用することで、混乱のない最高精度のパフォーマンスを維持します。

### 2. 対話による「テンプレート」の共創
* **一緒に作り上げる**
    あらかじめ用意された指示を一方的に与えるのではなく、OSとの対話を通じて、その時の目的に最適な「特化型テンプレート（指示の型）」をAIと一緒に作り上げます。

### 3. 柔軟な「遊び」による能力の解放
* **ガチガチに固めない**
    指示を厳格に固めすぎると、AIの柔軟性が失われます。あえて少し「緩さ（遊び）」を持たせた設計にすることで、AIが状況に応じて自由にそのポテンシャルを発揮できるようにしています。

## 🚀 使い方（導入方法）

お好みの方法で内容をコピー＆ペーストして適用してください。

1.  **導入:** チャット欄、System instructions、または Gem (カスタムGemini) に貼り付けます。
2.  **起動:** OSが起動したら、やりたいことを伝えます。
3.  **共創:** **OSそのものは書き換えずに**、対話しながら最適な「テンプレート」を作り上げ、実行してください。
